El Laboratorio de Neurociencias e Inteligencia Artificial Aplicada —**Neuro-IA LAB**— genera conocimiento mediante proyectos de investigación básica, avanzada y aplicada en Neurociencias e Inteligencia Artificial, con foco en el desarrollo de Interfaces Cerebro-Computadora.

El laboratorio participa en la formación de estudiantes de grado y posgrado, y ofrece servicios especializados a instituciones y organizaciones cuyos intereses se alinean con sus líneas de investigación.

El Neuro-IA LAB desarrolla proyectos de investigación básica, aplicada y de desarrollo tecnológico en interfaces cerebro-computadora, bases de datos multimodales de señales fisiológicas y procesamiento avanzado de señales EEG con inteligencia artificial.

### ✍️ Decodificación de escritura a mano alzada con EEG de alta densidad

El proyecto evalúa si es posible decodificar letras y trazos de escritura a mano alzada a partir de patrones de actividad cortical registrados con EEG de alta densidad. Se analizan señales durante escritura ejecutada e imaginada y se aplican técnicas avanzadas de procesamiento e inteligencia artificial para estimar la factibilidad de usar esta información en BCI de comunicación no invasivas.

---

### 🥽 Base de datos multimodal de estados mentales en realidad virtual

Desarrollo de un protocolo de adquisición de EEG, ECG, EMG y EOG en combinación con tareas en realidad virtual y aumentada, para conformar una base de datos multimodal de estados mentales como estrés, concentración, aburrimiento y estado de Flow.

---

### 🗣️ Base de datos multimodal de habla imaginada a partir de EEG

El proyecto tiene como objetivo principal la generación de una base de datos de señales EEG de alta densidad y audio durante tareas de habla ejecutada y habla imaginada. A partir de estas señales se estudiará la factibilidad de decodificar habla imaginada mediante modelos de ML e IA, con proyección al desarrollo de BCI de comunicación para personas con limitaciones motoras severas.
